FreeButWorseGPT is a simple and free AI chatbot that I made for fun :P

I wouldn't recommend it as an alternative to any other platform as the chatbot doesn't have any context.

## ‚ú® Features

- 100% free
- Saves your history locally
- Speech to text (write messages using your voice)
- Supports markdown
- Supports syntax highlighting
- Supports mermaid
- Supports LaTex
